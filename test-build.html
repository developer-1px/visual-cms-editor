Visual CMS Editor - Selection Test

🔧 Test Instructions:

  1. Open the built application in your browser: build/index.html
  2. Open Developer Tools (F12) and go to Console tab
  3. Try clicking on editable elements (headlines, buttons, text)
  4. Look for debug logs starting with 🔵, 🟢, 🟡, 🔴
  5. Check if data-selected attributes are being applied

🕵️ What to check:

  • Event firing: Look for "🔵 handleElementClick triggered" logs
  • Element detection: Check "hasEditable" and "hasRepeatable" values
  • Selection process: See "🟡 selectElement called" and attribute application
  • Visual feedback: Elements should get blue outline when selected

🔍 Manual Console Commands:

You can also test manually in the browser console:

// Check if editable elements exist
document.querySelectorAll('[data-editable]').length

// Manually apply selection to first editable element
const firstEditable = document.querySelector('[data-editable]');
if (firstEditable) {
    firstEditable.setAttribute('data-selected', 'true');
    firstEditable.setAttribute('data-selection-type', 'text');
    console.log('Applied selection manually');
}

// Clear all selections
document.querySelectorAll('[data-selected]').forEach(el => {
    el.removeAttribute('data-selected');
    el.removeAttribute('data-selection-type');
});
